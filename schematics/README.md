# Hardware Schematics

This folder contains the hardware schematics used for the **ECE 2021 Automatic Plant Wellness System** project.

**UNB Dev Board Schematic (Rev 4.1)**  
Used to interface moisture sensors, a water pump, LEDs, and audio output with the UNB Dev Board microcontroller.

These schematics were provided as part of the course hardware platform and are included here for reference and documentation purposes.
